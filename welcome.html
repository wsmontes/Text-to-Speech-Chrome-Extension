Welcome to Text-to-Speech Extension

Transform your browsing experience with easy text-to-speech functionality

Getting Started

The extension is already active! Try these methods to hear text spoken aloud:

  • Hover over text elements to hear them read (if hover mode is enabled)
  • Select any text and press Alt+S to have it read aloud
  • Right-click on selected text and choose "Speak selected text" from the context menu

Key Features

  • Multiple Activation Methods - Hover, keyboard shortcuts, or context menu
  • Voice Customization - Choose from available system voices
  • Speech Controls - Adjust rate, pitch, and volume
  • Site-Specific Settings - Enable or disable on specific websites
  • Reading Progress - Visual indication of current reading position

Keyboard Shortcuts

  • Alt+S - Speak selected text
  • Alt+X - Stop current speech
  • Alt+T - Toggle extension on/off

Important Notes

  • Voice quality depends on your browser and operating system
  • Some websites may restrict content selection or hovering
  • For best performance, use Chrome browser version 91 or higher
  • You can fine-tune settings from the extension options page
